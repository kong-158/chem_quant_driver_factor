# Changelog

## 0.1.0 - 2026-06-07

- 初始化化工行业 driver 因子研究 MVP。
- 支持 sample 数据自动生成。
- 支持 driver momentum、stock momentum、driver-stock gap 因子。
- 支持未来 20 日 / 60 日收益和超额收益标签。
- 支持 Rank IC、IC 汇总、月频分组回测和 Top-Bottom 绩效。
- 输出基础表格和图片。
